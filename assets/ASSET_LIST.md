# SlimeVenture Art Asset List

All assets should be PNG with transparency. Recommended sizes noted per category.
Place files in the corresponding subdirectory. The game will reference them via
`assets/<category>/<filename>.png`.

`def` already has an `emoji` field that can be replaced with a `sprite` path.
